## 已完成

### 总入口

- 侧边栏从多个入口整理为一个 Workspace2 总入口。
- 总入口内包含三个顶部标签：工作流、节点、模板。
- 支持快捷键：
  - `Shift+1` / `Shift+W`：工作流。
  - `Shift+2` / `Shift+N`：节点。
  - `Shift+3`：模板。

### Workflows2

- 使用 ComfyUI 官方默认工作流目录。
- 支持文件夹、子文件夹、重命名、拖拽整理。
- 支持拖拽到文件夹、展开文件夹区域和根目录。
- 支持插件回收站、恢复、移到系统回收站。
- Windows 下移到系统回收站已增加系统 API fallback；非 Windows 推荐安装 `send2trash`。
- 支持排序、自定义顺序、优先文件夹。
- 支持隐藏 `.json` 后缀。
- 支持搜索、清空搜索、点击名称打开工作流。
- 支持文件夹图标和颜色个性化。
- 支持 `Ctrl + 点击` 文件夹三角图标递归展开或折叠。
- 支持最近工作流历史记录。
- 最近工作流默认显示 5 条，设置里可调整 2-20 条，5 附近吸附。

### Nodes2

- 支持 Comfy、扩展节点、收藏节点分区。
- 扩展节点按插件来源整理。
- 支持节点收藏、收藏分组、收藏子分组。
- 支持收藏节点拖拽整理。
- 支持节点拖拽到画布。
- 支持点击节点后，在画布点击放置节点。
- 支持节点预览卡片。
- 支持模糊搜索和拼音搜索。
- 支持搜索清空按钮。
- 支持收藏节点备份与恢复。
- 支持从官方收藏导入 Nodes2。
- 支持从 Nodes2 导入官方收藏。
- 支持节点排序和自定义顺序。
- 支持缺失节点弱显示。
- 已修复语言包路径，避免 GitHub clone 后因目录名大小写不同显示 i18n key。
- 已增加 Nodes2 首屏快速显示：优先使用 ComfyUI 前端已注册节点。
- 已增加完整 `/object_info` 后台补全。
- 已增加 IndexedDB 节点缓存，刷新网页后可先显示上次缓存。

### Templates

- 已新增模板标签。
- 支持 `Alt+C` 保存选中节点为模板。
- 保存模板时保留节点相对位置和连接线。
- 支持模板拖拽到画布。
- 已修复模板拖拽到分组无效的问题。
- 支持模板右键菜单：
  - 重命名。
  - 放置到画布中心。
  - 复制模板名称。
  - 删除模板。
- 支持模板分组：
  - 父级分组。
  - 子级分组。
  - 新建分组。
  - 新建子分组。
  - 分组重命名。
  - 分组删除。
  - 模板拖拽到分组或根位置。
  - 分组拖拽到其它分组或根位置。
- 支持模板搜索。
- 支持模板字号滑块。
- 支持模板分组图标和颜色个性化基础能力。
- 支持模板排序。
- 支持模板 hover 预览，当前使用类似小地图的静态缩略预览。
- 支持单击模板进入待放置状态，再点击画布放置。
- 支持模板行右侧删除图标。
- 支持模板删除、模板分组删除的行内确认。

### 画布功能

- 支持 `Ctrl+G` 创建 Workspace2 编组。
- 支持 `Shift+G` 取消编组。
- 支持编组标题栏右键样式设置。
- 支持保存当前编组样式为默认样式。
- 已新增标题2节点。
- 标题2默认值：
  - 字号 24。
  - 透明 0。
  - 圆角 15。

### 全局体验

- Workflows2、Nodes2、Templates 的字号/行距滑块已统一底层计算逻辑。
- 同一滑块值现在会生成同一套基础字号、行高、padding 和间距。
- 三个标签页都保留“文件夹 / 分组字号大于普通项目字号”的层级区分。
- 字号滑块默认吸附点视觉标记已删除，仅保留数值吸附逻辑。
- 删除类危险操作第一批已从浏览器弹窗改为行内确认：
  - Workflows2 回收站单项移到系统回收站。
  - Workflows2 回收站清空到系统回收站。
  - Templates 删除模板。
  - Templates 删除分组。
  - Nodes2 删除收藏分组。
- 删除图标 hover 文案和危险色已统一：
  - 删除 / 移到回收站 / 移到系统回收站类图标统一使用危险 hover 色。
  - 模板分组、节点收藏分组删除图标已使用更明确的 hover 文案。

## 已知风险

- 主包插件数量很多，Nodes2 完整 `/object_info` 可能非常慢，当前已通过前端快速显示和 IndexedDB 缓存缓解。
