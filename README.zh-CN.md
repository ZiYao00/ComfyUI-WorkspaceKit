# ComfyUI-WorkspaceKit

[English](README.md) | 中文说明

> **维护说明：** 因出差原因，未来 2–3 周内我无法更新插件。请继续以公开测试版方式使用，并做好备份；Issue 和 PR 的回复可能会延后。

ComfyUI-WorkspaceKit 是一个面向 ComfyUI 的工作区增强插件，重点解决大型工作流、多插件节点库、节点收藏、模板复用和画布整理在日常使用中的管理问题。

它不是要替代 ComfyUI 官方界面，而是在官方功能之上补齐更适合重度用户的整理、搜索、复用和可恢复操作体验。

当前状态：**公开测试版，0.2.4**。它已经具备较完整的日常使用能力，但还不是稳定版 1.0。首次在主 ComfyUI 环境中使用前，请先备份工作流目录、用户设置、重要节点收藏和重要模板数据。

## 你可能需要它的情况

当你的 ComfyUI 使用时间变长后，通常会遇到这些问题：

- 工作流越来越多，官方列表不方便长期整理。
- 第三方插件越来越多，节点来源、分类和常用节点很难管理。
- 常用的节点组合经常重复搭建，缺少一键保存和复用的方式。
- 复杂工作流画布越来越乱，需要更清晰的编组、标题和注释结构。
- 删除、移动、重命名工作流时，希望有回收站和恢复机制，避免误操作。

WorkspaceKit 的目标，就是把这些高频整理动作集中到一个更顺手的工作区里。

## 主要功能

WorkspaceKit 目前提供一个统一侧边栏入口，入口内包含三个标签：

- **工作流2**：管理 ComfyUI 工作流目录中的工作流文件。
- **节点2**：浏览、搜索、收藏、分组和整理节点。
- **模板**：把已经连接好的选中节点保存为可复用模板，并支持分组管理。

同时，插件还包含两个画布辅助能力：

- **编组增强**：强化画布编组快捷键、标题栏样式设置和默认编组样式。
- **标题2**：用于给复杂工作流添加更清晰的视觉标题、区域说明和注释。

## 功能说明

### 工作流2

工作流2 面向“工作流文件管理”。它适合用来整理长期积累的大量 `.json` 工作流。

主要能力：

- 默认使用 ComfyUI 官方工作流目录。
- 树状文件夹管理，支持文件夹和子文件夹。
- 新建、重命名、拖拽移动工作流和文件夹。
- 支持拖到文件夹、展开的文件夹区域和根目录。
- 删除工作流时先进入 WorkspaceKit 回收站，可恢复后再决定是否移到系统回收站。
- 支持排序、自定义顺序、优先显示文件夹。
- 支持最近工作流历史记录，并可在设置中调整显示数量。
- 支持搜索、清空搜索、刷新和点击名称直接打开工作流。
- 支持打开工作流所在位置。
- 支持文件夹图标和颜色个性化。
- 支持 `Ctrl + 点击` 文件夹递归展开或折叠。

高级用户可以选择自定义工作流根目录。如果使用该功能，请只选择专门用于 ComfyUI 工作流的文件夹，不要选择磁盘根目录、桌面、下载目录或大型项目目录。

### 节点2

节点2 面向“节点查找、收藏和分组”。它适合安装了大量第三方节点、官方节点浏览器已经不够顺手的用户。

主要能力：

- 从 ComfyUI 当前节点信息中读取可用节点。
- 按 **Comfy** 和 **扩展** 区分节点来源。
- 扩展节点按插件来源整理，减少第三方节点混在官方分类里的情况。
- 支持节点搜索、模糊搜索和拼音搜索。
- 支持收藏节点、收藏分组和收藏子分组。
- 支持节点拖拽到收藏分组。
- 支持收藏节点在分组之间拖拽整理。
- 支持节点拖拽到画布。
- 支持点击节点后，再点击画布放置节点。
- 支持节点预览卡片。
- 支持与 ComfyUI 官方收藏进行导入 / 导出；写入官方收藏前会自动备份设置文件。
- 对缺失的第三方节点进行弱显示，不会静默删除。
- 支持节点缓存，便于大型节点库更快显示首屏内容。

### 模板

模板功能面向“节点组合复用”。它适合保存常用的节点小结构，例如一组已经连好的加载、预处理、控制、输出或后处理节点。

主要能力：

- 使用 `Alt+C` 将当前选中的节点保存为模板。
- 保存模板时保留节点相对位置和连接线。
- 支持模板拖拽到画布。
- 支持单击模板进入待放置状态，再点击画布放置。
- 支持模板分组和子分组。
- 支持模板搜索、排序、重命名和删除。
- 支持模板 hover 预览。
- 支持模板右键菜单，包括重命名、放置到画布中心、复制模板名称和删除。
- 支持模板删除和模板分组删除的行内确认。

测试阶段为保证兼容性，模板数据仍使用 ComfyUI 用户目录下现有的 Workspace2 兼容存储位置；升级到 WorkspaceKit 后，已有模板数据仍可继续使用。建议定期备份重要模板数据。

### 编组增强

编组增强面向“复杂画布整理”。它让 ComfyUI 的编组更接近常见设计软件的使用习惯。

主要能力：

- 使用 `Ctrl+G` 从选中节点创建 WorkspaceKit 编组。
- 使用 `Shift+G` 取消编组。
- 使用 `Shift + 左键` 切换 WorkspaceKit 编组是否忽略节点选择。
- 右击编组标题栏打开样式设置。
- 保存当前编组样式为默认预设之一。
- 支持设置编组边距、边框、阴影、动画等视觉参数。
- 编组数据保存到工作流中。

如果 `Ctrl+G` 与 ComfyUI 官方快捷键冲突，请先在 ComfyUI 快捷键设置中移除或修改官方绑定，或在 WorkspaceKit 设置中关闭相关开关。

### 标题2

标题2 是一个轻量的画布标题 / 注释节点，适合给大型工作流添加区域标题、说明和视觉分隔。

默认样式：

- 字号：24
- 背景：透明
- 圆角：15

## 快捷键

| 快捷键 | 功能 |
|---|---|
| `Shift+1` / `Shift+W` | 打开工作流2 |
| `Shift+2` / `Shift+N` | 打开节点2 |
| `Shift+3` | 打开模板 |
| `Alt+C` | 将当前选中节点保存为模板 |
| `Ctrl+G` | 创建 WorkspaceKit 编组 |
| `Shift+G` | 取消编组 |
| `Ctrl + 点击` 文件夹三角图标 | 递归展开或折叠文件夹 / 分组 |
| `Shift + 左键` | 切换 WorkspaceKit 编组是否忽略节点选择 |

## 安装

进入 ComfyUI 的 `custom_nodes` 目录：

```bash
cd ComfyUI/custom_nodes
git clone https://github.com/ZiYao00/ComfyUI-WorkspaceKit.git
```

然后重启 ComfyUI。

## 依赖

推荐安装：

```text
send2trash
```

请在 ComfyUI 使用的 Python 环境中安装依赖，用于更完整的跨平台系统回收站支持：

```bash
pip install -r requirements.txt
```

不要在不了解 ComfyUI Python 环境的情况下随意更新依赖。

## 系统回收站说明

WorkspaceKit 有两层回收站：

- **WorkspaceKit 回收站**：插件自己的可恢复回收站，用于工作流删除后的恢复。
- **系统回收站**：操作系统自己的回收站。

在 Windows 上，WorkspaceKit 已内置系统回收站 fallback。其它平台的系统回收站功能依赖 `send2trash` 以及当前桌面环境的支持。

## 首次使用前建议

WorkspaceKit 会移动、重命名和整理工作流文件。首次在主 ComfyUI 环境中使用前，建议备份：

- ComfyUI 工作流目录。
- ComfyUI 用户设置。
- 重要节点收藏数据。
- 重要模板数据。
